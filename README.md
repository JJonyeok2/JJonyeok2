### 🏋️ Atheleo (Main Project)

> 📱 카메라 기반 체형 분석 + AI 실시간 운동 피드백

- MediaPipe로 **실시간 키포인트 추출**
- OpenPose 기반 **체형 분류 및 데이터 분석**
- Django + FastAPI로 **AI 서버 구성**
- React Native + Expo로 **모바일 UI/UX 개발**
- AWS EC2, S3, Route 53으로 **서버 및 데이터 인프라 구축**
